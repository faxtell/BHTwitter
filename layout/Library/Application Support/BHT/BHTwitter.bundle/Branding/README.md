# Branding assets

Place your own legally usable branding images here before building the deb.

Recommended files:

- `twitter_logo.png` — transparent logo/wordmark used for in-app X logo replacements.
- `twitter_app_icon.png` — square icon used by the optional IPA patch script.
- `twitter_launch_logo.png` — transparent logo used by the runtime startup splash.
- `twitter_tab_logo.png` — optional small logo for tab/logo use.

Important:

This repository intentionally does not include Twitter/X-owned logo artwork. If you want the historical blue bird appearance, provide your own image files locally and make sure you have the right to use them.

If these PNG files are missing, the runtime branding patch generates a simple blue/white Twitter-style fallback image instead.
